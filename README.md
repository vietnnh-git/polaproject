# 🚀 PolaProject

A modern "Coming Soon" landing page built with HTML, CSS, and JavaScript.

## 📋 Features

- ✨ Beautiful gradient background with animations
- 📱 Fully responsive design (mobile, tablet, desktop)
- 📧 Email signup form
- 🎯 Progress bar animation
- 🔗 Social media links
- ⚡ Fast loading (no dependencies)
- 🔒 HTTPS enabled
- 🌍 Deployed on Vercel with CDN

## 🎨 Design

- **Color Scheme:** Purple gradient (#667eea to #764ba2)
- **Typography:** System fonts for best performance
- **Animations:** Smooth fade-in and bounce effects
- **Mobile First:** Optimized for all screen sizes

## 📁 Project Structure

```
polaproject/
├── index.html      # Main HTML structure
├── style.css       # All styling and animations
├── script.js       # Email form functionality
```

## 🚀 Quick Start

### Local Development

1. Clone this repository
```bash
git clone https://github.com/yourusername/polaproject.git
cd polaproject
```

2. Open `index.html` in your browser
```bash
# macOS/Linux
open index.html

# Windows
start index.html
```

### Deployment

This project is deployed on **Vercel**:
- Connected to GitHub for automatic deployments
- Any push to main branch automatically deploys
- Custom domain: polaproject.com

## 🔧 Customization

### Change the title and description
Edit `index.html`:
```html
<h1>Your Title</h1>
<p class="subtitle">Your Description</p>
```

### Change colors
Edit `style.css`:
```css
background: linear-gradient(135deg, #YOUR_COLOR1 0%, #YOUR_COLOR2 100%);
```

### Update social links
Edit `index.html`:
```html
<a href="https://your-social-link.com">Social</a>
```

### Update email contact
Edit `index.html`:
```html
<a href="mailto:your-email@example.com">Email</a>
```
